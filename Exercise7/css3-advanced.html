Container Queries

Rather than relying on 'Media Queries' we can directly apply container queries, they allow to modify an element based on the size of parent, unlike the media query which apply the changes based on the size of the viewport

Insert Property

his property replaces the four other properties: top, right, bottom, and left, allowing you to specify the inset of the positioned element on all four sides at once. The order is in clockwise order just like it would be for a margin or padding.
